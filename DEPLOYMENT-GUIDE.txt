================================================================================
TROUBLESHOOTING
================================================================================

Problem: "git: command not found"
Solution: Install Git from https://git-scm.com/download/win

Problem: GitHub Pages not showing my site
Solution:
- Wait 2-3 minutes and refresh
- Check Settings ‚Üí Pages for build status
- Verify branch is "main" and folder is "/ (root)"
- Check that repository is PUBLIC

Problem: Gumroad overlay not appearing
Solution:
- Verify Gumroad script is loaded (check browser console)
- Test in incognito mode (ad blockers may interfere)
- Verify buttons have class "gumroad-button"

Problem: Images not displaying
Solution:
- Check that /images/ folder was pushed to GitHub
- Verify image paths are correct (case-sensitive on GitHub)
- Check browser console for 404 errors

Problem: Can't push to GitHub
Solution:
- Create Personal Access Token (not password)
- Go to GitHub ‚Üí Settings ‚Üí Developer settings ‚Üí Personal access tokens
- Generate token with "repo" scope
- Use token as password when git asks
